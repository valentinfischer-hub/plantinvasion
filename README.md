# Plantinvasion

Mobile-Game-Projekt: 32-bit Idle-Breeder, Spieler zuechtet Pflanzen, kreuzt sie, sie kaempfen automatisch.

## Tech Stack
- Phaser 3 (Game Engine)
- Vite (Dev Server + Build)
- TypeScript
- Capacitor (spaeter, fuer Mobile-Wrapper)

## Setup

```bash
npm install
npm run dev
```

Dann oeffnen: http://localhost:5173

## Build

```bash
npm run build
```

Produziert statisches Build in `dist/`.

## Status

Phase 0: Skeleton steht. Naechster Schritt: Hello-World-Pflanze mit Wachstum und Ernte.
